# Surf Shack Landing

Landing page para membresías de Surf Shack - Club de Playa en Playa Caracol, Panamá.

## 🏗️ Stack

- React 18
- Vite
- TailwindCSS
- Framer Motion
- Wouter (routing)
- Lucide React (iconos)

## 🚀 Desarrollo

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```

## 🌐 Deploy

El sitio se despliega automáticamente en Render cuando se hace push a la rama `master`.

## 📝 Contenido

- **Hero**: Membresía Personal $25/mes + CTA WhatsApp
- **Beneficios**: Acceso ilimitado, descuentos, extras
- **Precios**: Personal ($25) vs Familiar ($40)
- **Testimoniales**: 3 reviews (dummies por ahora)
- **Galería**: 6 fotos del Drive
- **FAQ**: 8 preguntas frecuentes
- **CTA Final**: WhatsApp + opciones de pago
